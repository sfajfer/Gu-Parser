# Gu Parser

This project is a combination Java parser and front end to display the Gu after being parsed. It was originally written using Java Spring BOOT to connect to a MongoDB Atlas database, with the front end written in React.js. The database functionality still works, but due to the small size of the data the parser now uploads the data as well as exporting it to a json file.

The parser goes through a markdown file with specific markdown and LaTeX formatting and converts the stripped data into an object. The front end is included, but can also be found separately at https://github.com/sfajfer/Gu-SRD
